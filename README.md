# June

## CLI API

```sh
june generate <input.md> [-o public/output.html] [--light/--dark/--style ./custom.css] [--template ./template.html] [--ugc]
                                ^ give a default too   ^ switches theme                 ^ optional custom template    ^ sanitises markdown as UGC
```

```
june version
```
